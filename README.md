# How to use

## 1st. go to 'api' folder and run
    npm install
    npm start

## 2st. go to 'client' folder and run
    npm install
    npm start

### 3st. change the variable 'user' found in 'client/src/app.jsx' from 'true' to 'false' in order to see the login page
